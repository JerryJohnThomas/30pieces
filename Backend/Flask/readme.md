# Flask Backend

## how to install

* pip install Flask
* pip install Flask-Uploads
    - pip install flask-uploads
* pip install Flask-Runway
    * for continous reload
* pip install urllib3
* pip install os-sys
* pip install Werkzeug



## how to run

`python app.py`
By default, it runs on http://127.0.0.1:5000/ (localhost).

## Continuous development (chatgpt)
* set FLASK_ENV=development
* flask run
